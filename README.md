# python-web-bazel-bottle-hello-world

## Description
POC for bottle web framework with bazel.

## Tech stack
- python
  - bottle
- bazel

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
